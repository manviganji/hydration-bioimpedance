# Import libraries for data handling, math, and plotting
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Set a random seed so results are reproducible
np.random.seed(42)

# Define how many synthetic samples to generate
n_samples = 500

# Create a time axis to simulate continuous sensor recording
time = np.arange(n_samples)

# Define a water intake event (simulating when hydration begins)
water_intake_time = 250

# Define hydration state:
# 0 = dehydrated (before water intake)
# 1 = hydrated (after water intake)

# Hydration state is based on a simulated water intake event
hydration_state = np.where(time < water_intake_time, 0, 1)

# Create baseline bioimpedance trend over time
# Starts higher (dehydrated phase) and gradually decreases
bioz = 1200 - (time * 0.03)

# Add a gradual decrease in bioimpedance after hydration begins
# This simulates the physiological effect of water intake
bioz = bioz - np.where(
    time >= water_intake_time,
    40 * (1 - np.exp(-(time - water_intake_time) / 40)),
    0
)

# Add random noise to simulate real sensor variability
bioz = bioz + np.random.normal(0, 8, n_samples)

# Note: Time is included for visualization and structure,
# but bioimpedance (bioz) is the primary feature used for prediction

# Create a synthetic dataset
synthetic_df = pd.DataFrame({
    "time_s": time,
    "bioz": bioz,
    "hydration_state": hydration_state
})

Synthetic Data Generation Workflow¶

The synthetic bioimpedance dataset was generated through a step-by-step process designed to simulate realistic physiological behavior.

First, a time variable was created to represent continuous sensor recording. A hydration event was then defined to simulate when water intake occurs.

Based on this event, a hydration state was assigned, distinguishing between dehydrated and hydrated phases.

Next, baseline bioimpedance values were generated, starting at a higher level during dehydration. A gradual decrease in bioimpedance was then applied after the hydration event to reflect the physiological effect of increased body water.

To improve realism, controlled random noise was added to simulate variability in sensor measurements.

The final dataset was constructed by combining time, bioimpedance, and hydration state, which was then used for visualization and machine learning analysis.

Interpretation of the Bioimpedance Trend¶

The synthetic bioimpedance signal was designed to begin at a higher level during the dehydrated phase and then decrease gradually after the simulated water intake event. This reflects the expected physiological relationship between hydration and bioimpedance, where increased body water leads to lower electrical resistance.

The decrease in BioZ follows a gradual, non-linear trend rather than a perfectly linear or exponential curve, which better reflects how physiological changes in body hydration occur over time.

This behavior is based on the principle that increased body water improves electrical conductivity, resulting in lower impedance values.

The synthetic signal includes a gradual transition and controlled noise to simulate realistic sensor variability observed in physiological data. This helps the synthetic data resemble a real physiological signal rather than an idealized step change.

The purpose of this plot is to show the overall pattern and trend in bioimpedance over time, not to represent exact measured values. In this synthetic dataset, the main goal is to demonstrate that hydration-related changes in bioimpedance can be modeled in a controlled way.

Understanding Plot Signal Shape of the Bioimpedance Trend¶

The change in bioimpedance over time does not follow a perfectly linear or exponential mathematical pattern, but instead reflects a gradual, non-linear physiological response.

A linear trend would show a constant rate of decrease, resulting in a straight line. In contrast, an exponential decay curve would show a very rapid drop immediately after the hydration event, followed by a quick flattening of the curve.

In this synthetic dataset, the bioimpedance signal instead follows a gradual, non-linear decrease. This reflects a more realistic physiological response, where changes in body hydration occur progressively rather than instantaneously.

The inclusion of controlled noise further ensures that the signal does not appear overly smooth or idealized, making it more representative of real-world sensor data.

This approach emphasizes modeling realistic physiological behavior rather than fitting the signal to an idealized mathematical function.

Synthetic Data Generation¶

A synthetic dataset was generated to simulate bioimpedance behavior during dehydration and hydration phases.

The dataset includes: Time (time_s): represents a continuous recording Bioimpedance (bioz): simulated electrical resistance values Hydration state: labeled as 0 (dehydrated) and 1 (hydrated)

The hydration state is defined based on time, where the first half of the dataset represents dehydration and the second half represents hydration.

Bioimpedance values are generated such that: Higher values correspond to dehydration Lower values correspond to hydration

This reflects the physiological expectation that hydration affects electrical conductivity in the body.

Machine Learning Model Results¶

A Random Forest classifier was trained to predict hydration state using time and bioimpedance as input features.

The dataset was split into training and testing sets to evaluate model performance. The model achieved 100% accuracy, showing that the synthetic dataset contains a clear separation between dehydration and hydration conditions.

This means the selected features can detect hydration state in the synthetic data, although the result is expected to be very strong because the dataset was generated in a controlled way.

Additional Notes¶

In this synthetic dataset, hydration state is defined using time to simulate a controlled experiment. This makes the dataset useful for testing the machine learning workflow, but it is simpler than real-world hydration data.

In actual sensor data, hydration is not determined directly by time. Instead, it is reflected through physiological changes in bioimpedance.
